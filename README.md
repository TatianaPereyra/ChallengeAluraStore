# ChallengeAluraStore
Challenge de Alura sobre analisis de datos con python.

## Archivo Colab
Se trabajo con la plataforma Google Colab con un conjunto de datos ya 
cargados previamente sobre 4 tiendas diferentes.
El objetivo del trabajo de analisis fue considerar ciertor rubros y, en base a la comparacion de estos datos, recomendar la tienda menos eficiente para su venta.

Se utilizo el lenguaje de python y librerias relacionadas (tales como Matplotlib y Pandas) para 
la manipulacion y analisis de estos datos cargados.

## Informe
Finalmente, se realizo un informe agrupando los diferentes graficos realizados en el analisis
aclarando cual tienda se recomienda vender y el porqué detrás de la desición.
